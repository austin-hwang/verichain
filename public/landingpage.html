Enabling the Dissemination of High-Value IoT Data Through Decentralized Auctions on the Blockchain

Verichain is a decentralized, open data marketplace that allows data consumers to place bids on auctions for high-value IoT devices to compensate for invaluable data.

Goals

Establish a Proof of Valid Data

  • Submit hash of first 256 bytes of data, which is then verified by the smart contract with the hash sent by the IoT device on instantiation.
  • If data is not verified within certain time slot, then both the data consumer and provider gets a strike.
  • Strikes are weighted based on reputation to correctly proportion the consequences.

Build a Robust Data Marketplace and Incentivize Users

  • Create a marketplace that facilitates batch purchasing of data from IoT end users
  • Target high value IoT device users (e.g. smart speakers) generating high-value, exclusive data that non-corporate consumers currently cannot access due to corporate data monopolization
  • Establish user trust by implementing reputation system to economically discourage dishonest players

Create Reliable Reputation and Feedback System

  • Require registration based on unique UID on blockchain
  • Each IoT device will be associated with its owner's UID
  • Reputation is automatically assigned based on success of auction to prevent user manipulation and collusion
  • Strikes assigned based on failure to provide valid data or failure to validate data.

Core Values

Democracy

Users should have a say in what data they contribute.

Privacy

Preventing large conglomerates from getting data for free.

Security

Allow secure transfer of data API keys and prevent data fraud.

Scalability

Ensure platform scaling proportional to IoT device increase.

Key Features

View Device Metadata

Data consumers can view what type of device is selling data, what kind of data is being sold, and location of device.

Batch Requets

Data consumers can batch thousands or even millions of auctions of IoT devices to buy large quantities of similar device or locational data.

Filter Devices

Data consumers can filter auctions by what kind of data or device that they want (e.g smart fridges, smart homes).

Why the Blockchain?

1. Transparency: Ethereum Public Ledger to authenticate users and view all app transactions
2. Censorship Resistant: Centralized corporations cannot arbitrarily decide what constitutes free speech
3. Eternal Persistency: Inter-planetary file system architecture distributes and persists content forever
4. Token Functionality: Blockchain enables payouts, autonomous platform, and token-based transactions
5. Extended Ecosystem: Long-term vision to tokenize user data, exclusive marketplace, and developer platform
